# Облачное хранилище документов с автоматической категоризацией

Веб‑приложение на Node.js и Express для хранения документов. Файлы загружаются на сервер, информация о них сохраняется в базе данных SQLite, а категория документа определяется автоматически по имени и типу файла. Пользователь может изменять категорию и добавлять описание.

## Технологии

- **Node.js** (Express) — HTTP‑сервер и REST API
- **Multer** — загрузка файлов на сервер
- **SQLite (better‑sqlite3)** — встроенная база данных для метаданных документов
- **Bootstrap 5** — современный интерфейс без тяжёлого фронтенд‑фреймворка

## Запуск проекта

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Запустите сервер:

   ```bash
   npm run dev
   ```

   или

   ```bash
   npm start
   ```

3. Откройте в браузере:

   `http://localhost:3000`

## Функциональные возможности

- загрузка одного или нескольких файлов;
- автоматическое определение категории документа по:
  - расширению файла (изображения, таблицы и т.п.),
  - ключевым словам в имени (учёба, работа, финансы, личные документы);
- просмотр списка документов с:
  - фильтрацией по категории,
  - поиском по названию, ключевым словам и описанию;
- редактирование карточки документа:
  - изменение категории,
  - добавление/редактирование текстового описания;
- удаление документа (из БД и с диска);
- открытие загруженного файла в новой вкладке.

## Структура проекта

- `src/server.js` — настройка Express, маршруты API, раздача статических файлов;
- `src/db.js` — инициализация SQLite и функции работы с таблицей `documents`;
- `src/classifier.js` — логика автоматической категоризации по имени/типу файла;
- `public/index.html`, `public/styles.css`, `public/app.js` — одностраничный веб‑интерфейс;
- каталог `uploads/` — сами загруженные файлы;
- каталог `data/` — файл базы данных `storage.db`.
